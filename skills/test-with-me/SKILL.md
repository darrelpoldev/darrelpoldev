# Test With Me

- Purpose: make sure the reviewed code changes are FULLY tested
- Assess which kinds of testing the change needs — not every change needs all three
- Propose a test plan: which of /unit-test-code, /e2e-test-code, /regression-test-code apply and why
- Run them once the user confirms the plan
- Summarize overall test status: what's covered, what's not, what's risky
